## 1. 공통 레이아웃

애플리케이션 전반에 적용되는 기본 구조입니다. 데스크톱에서는 좌측 사이드바, 모바일에서는 하단 탭 또는 햄버거 메뉴를 사용합니다.

### 1.1 데스크톱 레이아웃

```text
+---------------------------------------------------------------+
|  [Logo] IGK-TodoList                  [User Profile] [Logout] |  <-- Header
+-----------------------+---------------------------------------+
| [Sidebar]             | [Main Content Area]                   |
|                       |                                       |
| - 할일 목록 (Home)    |                                       |
| - 휴지통 (Trash)      |                                       |
| - 국경일 (Holidays)   |                                       |
| - 설정 (Profile)      |                                       |
|                       |                                       |
|                       |                                       |
|                       |                                       |
+-----------------------+---------------------------------------+
```

### 1.2 모바일 레이아웃

```text
+---------------------------------------+
| [Menu] IGK-TodoList            [User] | <-- Header
+---------------------------------------+
|                                       |
| [Main Content Area]                   |
|                                       |
|                                       |
|                                       |
+---------------------------------------+
| [Home]  [Trash]  [Holiday]  [Profile] | <-- Bottom Nav
+---------------------------------------+
```

## 3. 메인 할일 목록 화면

**참조**: PRD 10.1, 시나리오 2.1.1

```text
+---------------------------------------------------------------+
|  [Header]                                                     |
+-----------------------+---------------------------------------+
| [Sidebar]             |                                       |
| > 할일 목록           |  <h2>나의 할일</h2>                   |
|   휴지통              |                                       |
|   국경일              |  [ 검색창 (제목/내용) ] [필터▼] [정렬▼] |
|   설정                |                                       |
|                       |  +---------------------------------+  |
|                       |  | [v] [긴급] 프로젝트 보고서 제출 |  |
|                       |  |     2025-11-25 (오늘)           |  |
|                       |  |     [수정] [삭제]               |  |
|                       |  +---------------------------------+  |
|                       |                                       |
|                       |  +---------------------------------+  |
|                       |  | [ ] 영어 과제                   |  |
|                       |  |     2025-11-28까지              |  |
|                       |  |     [수정] [삭제]               |  |
|                       |  +---------------------------------+  |
|                       |                                       |
|                       |  +---------------------------------+  |
|                       |  | [ ] 중간고사 준비               |  |
|                       |  |     2025-12-01 ~ 2025-12-05     |  |
|                       |  |     [수정] [삭제]               |  |
|                       |  +---------------------------------+  |
|                       |                                       |
|                       |          [ + 할일 추가 버튼 ]         |
+-----------------------+---------------------------------------+
```

**UI 요소 상세**:
- **필터**: 전체 / 진행중 / 완료됨
- **정렬**: 마감일순 / 생성일순
- **카드 스타일**:
    - 완료된 항목: 텍스트 취소선, 회색 처리
    - 마감 임박(오늘/지남): 날짜 텍스트 빨간색 강조
- **FAB (+)**: 모바일에서는 우측 하단 플로팅 버튼

## 4. 할일 추가/수정 모달

**참조**: PRD 5.1.2, ERD Todo

```text
+-----------------------------------------+
|  할일 추가 / 수정                   [X] |
+-----------------------------------------+
|                                         |
|  제목 * (title)                         |
|  [___________________________________]  |
|                                         |
|  상세 내용 (content)                    |
|  [                                   ]  |
|  [                                   ]  |
|  [___________________________________]  |
|                                         |
|  시작일 (startDate)   만료일 (dueDate)  |
|  [ YYYY-MM-DD ]       [ YYYY-MM-DD ]    |
|                                         |
|  +-----------------------------------+  |
|  | [ 취소 ]               [ 저장 ]   |  |
|  +-----------------------------------+  |
+-----------------------------------------+
```

**인터랙션**:
- 저장 버튼 클릭 시 `startDate > dueDate`일 경우 에러 메시지 표시 (붉은색 텍스트)
- 제목 미입력 시 에러 표시

## 5. 휴지통 화면

**참조**: PRD 5.1.3, 시나리오 2.1.3

```text
+---------------------------------------------------------------+
|  [Header]                                                     |
+-----------------------+---------------------------------------+
| [Sidebar]             |                                       |
|   할일 목록           |  <h2>휴지통 🗑️</h2>                   |
| > 휴지통              |  <span color="gray">삭제된 할일은 복구 가능합니다.</span> |
|   국경일              |                                       |
|   설정                |  +---------------------------------+  |
|                       |  | [삭제됨] 주간 보고서 작성       |  |
|                       |  | 삭제일: 2025-11-25 15:00        |  |
|                       |  | [복원 ↺] [영구삭제 X]           |  |
|                       |  +---------------------------------+  |
|                       |                                       |
|                       |  +---------------------------------+  |
|                       |  | [삭제됨] 옛날 할일              |  |
|                       |  | 삭제일: 2025-10-01              |  |
|                       |  | [복원 ↺] [영구삭제 X]           |  |
|                       |  +---------------------------------+  |
|                       |                                       |
+-----------------------+---------------------------------------+
```

**기능**:
- **복원**: `status`를 `active`로 변경, `deletedAt` 제거
- **영구삭제**: DB에서 `DELETE` 수행 (경고창 팝업 필수)
